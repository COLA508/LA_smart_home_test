功能说明
===========================

门窗控制
-------------------
- 舵机控制门（0°关，135°开，5 秒后自动关闭）
- 舵机控制窗户（雨天自动关，晴天自动开）

环境检测
-------------------
- DHT11 检测温湿度
- 光照传感器+PIR 联动开关 LED
- 雨滴传感器控制窗户
- 温度超过阈值时自动开风扇

灯光系统
-------------------
- WS2812 RGB 灯带
  - 关闭模式
  - 自动模式（颜色渐变）
  - 手动模式（网页/语音设置 RGB 值）
- 按钮控制 RGB 模式切换

语音识别
-------------------
- “开门 / 关门”
- “开灯 / 关灯”
- “开窗 / 关窗”
- “开风扇 / 关风扇”
- “RGB 开启 / 关闭”

RFID 刷卡开门
-------------------
- 支持 RC522 模块识别卡片开门
