接线说明
===========================

以下是主要的模块接线：

- **门舵机** → GPIO13
- **窗舵机** → GPIO4
- **DHT11** → GPIO15
- **光照传感器** → GPIO34
- **PIR 红外传感器** → GPIO33
- **LED** → GPIO26
- **风扇** → GPIO27
- **雨滴传感器** → GPIO35
- **RGB 灯带** → GPIO5
- **RGB 控制按钮** → GPIO32
- **语音模块** → RX=19, TX=18
- **LCD1602** → I2C (SDA=21, SCL=22)
- **RC522 RFID** → I2C (0x28)
